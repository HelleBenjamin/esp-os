This file contains documentation of available terminal commands
- help: display a list of available commands
- sysinfo: display system information
- echo <text> : repeats a text
- pinset <pin> <value>: set pin(io pin 1, 2 and onboard led(3)) state to 1 or 0
- pinread <pin>: reads pin(io pin 3 and 4) state
- calculator: a simple calculator
- my_program: an example program in programs.cpp
- ram <from> <to>: read ram from start addr to end addr
- reboot: restart the os
- pinreset: set all output pins to 0
- direct_syscall, dscall: starts a direct syscall program
- get_eeprom: read eeprom
- tasks: list all tasks
- taskill <pid>: kill a task
